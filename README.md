## About the Name
- *Laurus* is a genus of evergreen trees or shrubs belonging to the plant family Lauraceae.
- Bay laurel (*Laurus nobilis*) can be used to make laurel wreaths, which were given to winners of some athletic games like the ancient Olympics. As users can compete with others for victory in the game, we name it *Laurus*, a symbol of victory and competition.
- Don't forget the word 'laureate', which means 'crowned with laurels', in 'poet laureate', 'baccalaureate' and 'Nobel laureate'. This shows the connection between *Laurus* and wisdom, knowledge, exactly what you can gain by playing the game.
